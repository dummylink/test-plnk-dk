Content of GPL-Addon
---------------------

The GPL-Addon is an extra package for openPOWERLINK, that contains sources released
under GPL license.

- Ethernet driver for netX-500 under Linux
  (directory Edrv/netx_ethhub/source/EdrvNetx.c)

- Demo project with readme.txt for netX-500 under Linux
  (directory Examples/netx500/Linux/gnu/demo_mn_kernel)



Installation
-------------

Extract the GPL-Addon archive to the same directory as the openPOWERLINK archive.
Use the very same version of both.


Documentation
--------------

The documentation of the openPOWERLINK protocol stack can be downloaded from

    http://www.systec-electronic.com/html/index.pl/en_download_OpenPOWERLINK

    It contains an introduction and a reference manual. A registration is
    required for downloading.
